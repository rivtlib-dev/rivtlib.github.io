**rivtlib** is an open source Python package for writing, sharing and modifying
engineering documents with minimum friction. It implements a markup up language
and open source framework for editing and publishing (see `rivt.info
<https://rivt.info>`_). The source code is here `GitHub
<https://github.com/rivtlib-dev/rivtlib/>`_.
